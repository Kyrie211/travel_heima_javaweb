# travel_heima_javaweb
本项目采用Idea工具编码。是一个JavaWeb练手项目（黑马），该项目整合了前端三大件、jQuery、ajax、bootstrap、Servlet、filter、mysql、jdbc（Spring Jdbc、Druid连接池）、redis缓存（Java中使用jedis）等技术。
